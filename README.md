# nvim

My simple Neovim config.

## Specs

- **LSP** - `nvim-lspconfig` + `mason` 
- **Completion** - `blink.cmp`
- **Formatting** - `conform.nvim`
- **Syntax** - Tree-sitter highlighting
- **Navigation** - `fzf-lua` + `flash.nvim`
- **File tree** - `oil.nvim`
- **Git** - `gitsigns`
- **Outline** - `aerial.nvim`
- **Theme** - [Kanagawa](https://github.com/rebelot/kanagawa.nvim)

## Key Mappings

Leader key: `<Space>`

## Requirements

- Neovim >= 0.10
- [lazy.nvim](https://github.com/folke/lazy.nvim) (auto-installed)
- A [Nerd Font](https://www.nerdfonts.com/) for icons
